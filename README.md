## Features

✔️ Modular Abilities

✔️ FPS Character Controller and non FPS Character Controller

✔️ Smooth Movement (Acceleration and Deceleration)

✔️ Handles Slopes

✔️ Air Control

✔️ Easily Adjustable from the Inspector

✔️ Sprint Ability

✔️ Crouch Ability

✔️ Fly Mode Ability

✔️ Jump Ability

🔨 Swim Ability.
